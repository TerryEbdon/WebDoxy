# WebDoxy

~~~{.txt}
usage: Build -[bcdghjntv] [project-name]*
 -b,--backup          Backup configuration & source files for all projects
 -c,--create          Create configuration file and source folders for
                      project(s)
 -d,--date <arg>      Optional date for journal entry. Today if not
                      specified
 -g,--generate        Generate web site(s) for project(s)
 -h,--help            Show usage information
 -j,--journal         Create a daily diary entry
 -n,--number <arg>    number of days to generate
 -p,--project <arg>   project name, for stubs command
 -s,--stub            generate stub files for given project
 -t,--toc             Create a table of contents page, referencing all
                      configured projects
 -v,--validate        Run a validation check on the specified projects
 -w,--week            Create weekly diary page
 ~~~
